FILTERING RESULTS

TEXT VALUES

The WHERE clause is used to extract only those records that fulfil a specified criterion

E.g: SELECT column_list FROM table_name WHERE condition;

For text columns, surround any text that appears in the statement with single quotation marks

The following statement selects all records in which the concession card state is equal to 'Active'

SELECT * FROM concession_cards WHERE concession_cards.state = 'active'



COMPARISON OPERATORS

The following comparison operators can be used in the WHERE clause

= Equal

!= Not equal

> Greater than

< Less than

>= Greater than or equal

<= Less than or equal

BETWEEN Between an inclusive range

So for example, to display all customers names listed in the table, with the exception of the one with id 5:

SELECT * FROM customers WHERE id != 5;



MULTIPLE COMPARISONS

The AND operator can be used for multiple comparison operators

The results given will satisfy both the comparison operators

For example, the below query is to find the name of the customers between 30 to 40 years of age

SELECT id, FirstName, LastName, Age FROM customers WHERE Age >= 30 AND Age <=40;
