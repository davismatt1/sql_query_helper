SORTING RESULTS

ORDERING RESULTS

To sort the returned data, the query can use the ORDER BY statement

The ORDER BY keyword sorts the results in ascending order

The following example sorts the concession cards table by the state column:

SELECT * FROM concession_cards ORDER BY state;



MULTIPLE ORDERING

ORDER BY can sort retrieved data by multiple columns

Use commas between the columns, it will order the results in the order columns are listed

The following example sorts the concession cards table by the state column, and then if there are any duplicated concession card states, they will by ordered by the customer_id column:

SELECT * FROM concession_cards ORDER BY state, customer_id;



CONCATENATION

The CONCAT function is used to concatenate two or more text values and returns the concatenating string

Taking the above example, if we wanted to concatenate the state with the customer_id, separating with a comma

The result could look like this: active, 123456789. This query is below:

SELECT CONCAT(state, ‘,’, customer_id) FROM concession_cards;
